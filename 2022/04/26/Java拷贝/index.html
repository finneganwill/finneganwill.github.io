1. Java中创建对象的5种方式

1.1 new

new 关键字创建对象是最常规的一种方法，可以通过调用有参或无参的构造方法实现对象的创建，例如：

1.2 clazz.newInstance()

通过 Class 类的API实现创建对象，是一种反射方式。这种方式调用的是 public 无参构造，因此$\textcolor{red}{类必须具有公有的无参构造}$，否则会报错。

这种方式在 Java 9 之后已经不推荐使用了：

1.3 constructor.newInstance()

java.lang.reflect.Constructor 类中也有一个 newInstance() 方法可以创建对象，和上一个方法一样，也是一种反射方式。通过这个方法$\textcolor{red}{可以调用无、有参和公、私有的构造方法} $。

实际上 clazz.newInstance() 内部仍然是调用的 constructor.newInstance()

1.4 Clone

Object类中提供了一个 native 的 clone() 方法，A.clone() 会创建一个内容和对象 A 相同的对象，使用该方式==不会调用构造方法==。

protected native Object clone() throws CloneNotSupportedException;

1.5 反序列化

反序列化是把磁盘文件中的对象数据或者把网络节点上的对象数据，恢复成Java对象模型的过程，反序列化==不会调用任何构造函数==。

2. 对象拷贝

2.1 引用拷贝

引用拷贝：创建一个引用，该引用与原引用都指向同一个对象。

2.2 浅拷贝

浅拷贝：创建一个引用，并对对象进行拷贝，将创建的引用指向拷贝对象，拷贝时：

  • 基本数据类型的字段进行值复制
  • 引用类型的字段复制一份引用并指向原引用所指向的对象

2.3 深拷贝

深拷贝：创建一个引用，并对对象进行拷贝，将创建的引用指向拷贝对象，拷贝时：

  • 基本数据类型的字段进行值复制
  • 引用类型的字段将引用和引用的对象都复制一份，并将拷贝引用指向拷贝对象

3 总结

  1. java中创建对象的几种方式

    • new
    • clazz.newInstance(); // java 9 开始弃用
    • constructor.newInstance();
    • obj.clone(); // 默认浅拷贝
    • deseriallization

  2. 拷贝

    • 引用拷贝：创建一个引用指向原对象
    • 浅拷贝：基本类型进行值复制；引用类型只复制引用，不复制引用的对象；clone()方法默认采用浅拷贝实现
    • 深拷贝：基本类型进行值复制；引用类型既复制引用，也复制引用的对象
